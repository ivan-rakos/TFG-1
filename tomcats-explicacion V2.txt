Vamos a utilizar el driver macvlan para poder crear una red en docker que sea igual que la de nuestro host.
Macvlan no permite conectarte a los dockers desde el host que tiene docker, por lo tanto el siguiente proceso lo hacemos desde una maquina virtual.
Antes de nada tenemos que poner la tarjeta de red en modo promiscuo para que acepte todos los paquetes --> sudo ifconfig enp0s3 promisc

Tenemos que crear una red en docker y para ello utilizamos el comando:
docker network create -d macvlan --subnet=192.168.1.0/24 --gateway=192.168.1.1 -o parent=enp0s3 public

Seguidamente creamos los dockers:
docker run -dit --net=public --name tomcat tomcat

Ahora desde el host que tiene los dockers ya podemos acceder a los contenedores
